## Introduction

React 기반으로 만들어진 밴드 DAY6 팬 페이지

## Features

- Day6 멤버 소개
- Day6 및 솔로 뮤직 비디오 제공

- Day6 및 개인 유튜브 공식 계정의 재생목록 제공
